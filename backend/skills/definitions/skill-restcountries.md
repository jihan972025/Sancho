### restcountries — Country Information

Get detailed country info — capital, population, languages, currencies, borders, and more.

**Parameters:**
- `name` (string, required): Country name (e.g., "South Korea", "Japan", "France")

**Examples:**
```
[SKILL_CALL]{"skill": "restcountries", "params": {"name": "South Korea"}}[/SKILL_CALL]
```
```
[SKILL_CALL]{"skill": "restcountries", "params": {"name": "Germany"}}[/SKILL_CALL]
```
