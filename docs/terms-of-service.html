1. 서비스 개요

Sancho(이하 "앱")는 오픈소스 AI 비서 데스크톱 애플리케이션입니다. 본 약관은 앱의 사용 조건을 규정합니다.

2. 서비스 이용 조건

  • 앱은 무료로 제공되며, 오픈소스 라이선스에 따라 배포됩니다.
  • 사용자는 관련 법률을 준수하여 앱을 사용해야 합니다.
  • 13세 미만의 사용자는 보호자의 동의 하에 사용해야 합니다.

3. Google 계정 연동

  • 앱은 Google OAuth 2.0을 통해 사용자 인증을 제공합니다.
  • Google 계정 연동은 선택 사항이며, 연동 없이도 핵심 기능을 사용할 수 있습니다.
  • Google 계정 사용 시 Google 서비스 약관이 적용됩니다.

4. 사용자 데이터

  • 모든 사용자 데이터는 사용자의 로컬 컴퓨터에만 저장됩니다.
  • 앱은 사용자 데이터를 외부 서버로 전송하지 않습니다.
  • 데이터 관리에 대한 자세한 내용은 개인정보처리방침을 참조하세요.

5. API 키 및 외부 서비스

  • 앱은 다양한 LLM 제공자(OpenAI, Anthropic, Google 등)의 API를 사용합니다.
  • 사용자는 자신의 API 키를 직접 입력하여 사용하며, 이에 따른 비용은 사용자 부담입니다.
  • 외부 API 서비스의 이용약관은 해당 서비스 제공자의 약관을 따릅니다.

6. 면책 조항

  • 앱은 "있는 그대로(AS-IS)" 제공되며, 어떠한 명시적 또는 묵시적 보증도 하지 않습니다.
  • AI가 생성한 응답의 정확성, 완전성, 적합성에 대해 보증하지 않습니다.
  • 앱 사용으로 인한 직접적 또는 간접적 손해에 대해 책임지지 않습니다.
  • 암호화폐 분석 기능은 정보 제공 목적이며, 투자 조언으로 간주해서는 안 됩니다.

7. 지적 재산권

  • 앱은 오픈소스로 배포되며, 해당 라이선스 조건을 따릅니다.
  • 사용자가 생성한 콘텐츠에 대한 권리는 사용자에게 있습니다.

8. 서비스 변경 및 중단

개발자는 사전 통지 없이 앱의 기능을 변경, 추가 또는 제거할 수 있습니다. 오픈소스 특성상 사용자는 이전 버전을 계속 사용할 수 있습니다.

9. 약관 변경

본 약관은 변경될 수 있으며, 변경 시 이 페이지를 통해 공지합니다. 변경 후에도 앱을 계속 사용하는 것은 변경된 약관에 동의하는 것으로 간주합니다.
