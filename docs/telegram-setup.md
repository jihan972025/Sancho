# Telegram Setup Guide

## Step 1: Get API Credentials (one-time only)

1. Visit https://my.telegram.org on your phone's browser
2. Log in with the phone number registered to your Telegram account
3. Click "API development tools"
4. Enter any app name and a short description, then create
5. Copy the API ID (number) and API Hash (alphanumeric string)

## Step 2: Sancho Settings

1. Launch Sancho
2. Go to Settings > Chat App tab
3. Check "Enabled" in the Telegram section
4. Enter your API ID and API Hash
5. Click Save Settings

## Step 3: Connect via QR Code

1. Click the Connect button
2. A QR code will appear on screen
3. On your phone, open Telegram > Settings > Devices > Link Desktop Device > Scan the QR code
4. Once the status changes to "Connected", you're all set

## Step 4: Usage

- Send a message to "Saved Messages" (message yourself) in the Telegram app
- Sancho will send the LLM response to both the chat window and Telegram
- Telegram messages appear as blue bubbles in the chat window
