# WhatsApp Setup Guide

## Step 1: Sancho Settings

1. Launch Sancho
2. Go to Settings > Chat App tab
3. Check "Enabled" in the WhatsApp section

## Step 2: Connect via QR Code

1. Click the Connect button
2. A QR code will appear on screen
3. On your phone, open WhatsApp > Settings > Linked Devices > Link a Device > Scan the QR code
4. Once the status changes to "Connected", you're all set

## Step 3: Usage

- Send a message to yourself in WhatsApp (Message yourself)
- Sancho will send the LLM response to both the chat window and WhatsApp
- WhatsApp messages appear as green bubbles in the chat window

## Notes

- **No API key required**: WhatsApp connects using only a QR code.
- **Persistent session**: Once connected, the session is maintained. If disconnected, it will auto-reconnect.
- **WhatsApp Web Version**: If you get a 405 error when connecting, try changing the version value in Settings.
